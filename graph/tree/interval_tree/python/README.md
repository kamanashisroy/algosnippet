
This interval tree is based on the book *Introduction to Algorithms*.


```

                < b [a ... b] a < 
             /|                   |\
            / |                   | \
           /  |                   |  \
          / L |                   | R \
         /    |                   |    \
        -------                   -------
```

The interval tree above has a root that has an interval from a to b.
